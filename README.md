# Coastguard App

Aplicación móvil y web desarrollada con **React + Vite + Capacitor**, diseñada para gestionar propiedades, clientes y operaciones internas de Coastguard Homes.

---

## 🚀 Tecnologías principales

- React 18
- Vite 5
- Capacitor 6
- Android SDK
- Java 17
- GitHub Actions (CI/CD)

---
